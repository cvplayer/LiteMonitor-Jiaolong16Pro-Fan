# 让任务栏显示风扇转速 —— 小白教程

## 你需要准备

- 一台机械革命蛟龙 16 Pro 笔记本
- Windows 11 系统
- 5 分钟时间

---

## 第一步：下载文件

打开这个 GitHub 地址（你上传之后把链接替换到这里）：

```
https://github.com/你的用户名/仓库名
```

点击绿色的 **Code** 按钮 → **Download ZIP**。

下载完把 ZIP 解压到一个文件夹，比如 `D:\风扇插件\`。

---

## 第二步：安装风扇桥

1. 打开解压的文件夹，找到 `ec-bridge` 文件夹
2. 双击里面的 `ec-bridge.exe`
3. 如果弹出「Windows 保护了你的电脑」，点「更多信息」→「仍要运行」

**验证**：打开浏览器，输入 `http://localhost:18900/fan`

如果看到类似这样的字：
```
{"cpu_fan":2560,"gpu_fan":2414,"ok":true}
```
说明桥安装成功。关掉浏览器。

---

## 第三步：安装 LiteMonitor 插件

1. 找到 LiteMonitor 的安装文件夹，进入 `resources\plugins\` 目录
2. 把解压出的 `FanSpeed.json` 文件**复制**到这个目录里

---

## 第四步：让风扇数字出现在任务栏

1. 打开 LiteMonitor
2. 右键任务栏的 LiteMonitor 图标 → 设置
3. 点击「重载插件」
4. 在插件列表里找到「风扇转速」→ 勾选启用

任务栏上就会显示：
```
CPU风扇 2560 RPM  │  GPU风扇 2414 RPM
```

---

## 第五步：让桥开机自动启动（不用每次双击）

1. 按键盘 `Win + R`，输入 `shell:startup`，回车
2. 把解压出的 `start-bridge.vbs` **复制**到弹出的文件夹里

以后每次开机，桥会自动在后台静默启动，没有任何窗口。

---

## 常见问题

**Q: 浏览器打不开 `localhost:18900/fan`？**
A: 桥没启动。双击 `ec-bridge.exe` 再试。

**Q: LiteMonitor 显示 ❌ 或 ❓？**
A: 说明桥没在运行。双击 `ec-bridge.exe`，然后在 LiteMonitor 设置里重载插件。

**Q: 风扇数字不变化？**
A: 待机时风扇转速本来就不变。打开一个游戏试试，数字应该会涨。

**Q: 任务栏数字是什么颜色？**
A: 绿色（3000 转以下安静）、黄色（3000-4500 转有点响）、红色（4500 转以上起飞）。

**Q: 这个会不会影响电脑？**
A: 不会。它只读取风扇转速，不修改任何系统设置。
